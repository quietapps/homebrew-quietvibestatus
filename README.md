# Quiet Vibe Status Homebrew tap

Homebrew tap for [Quiet Vibe Status](https://github.com/quietapps/QuietVibeStatus) — a native macOS menu bar app that watches every AI coding agent you have running from the Mac's notch, and lets you approve permissions and answer questions without leaving the window you're in.

## Install

```sh
brew tap quietapps/quietvibestatus
brew install --cask quietvibestatus
```

## Update

```sh
brew update
brew upgrade --cask quietvibestatus
```

## Uninstall

```sh
brew uninstall --cask quietvibestatus
brew untap quietapps/quietvibestatus
```
